# 🎲 JavaScript Dice Roller App

A simple and interactive Dice Roller application built using **HTML**, **CSS**, and **JavaScript**. This project generates random dice rolls, displays the corresponding dice images, and plays a dice roll sound effect for a more engaging experience.

## 🚀 Features

* Roll one or multiple dice at once
* Generates random results using JavaScript
* Displays dice images corresponding to each roll
* Dice roll sound effect
* Clean and responsive user interface
* Beginner-friendly project demonstrating DOM manipulation and event handling

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
